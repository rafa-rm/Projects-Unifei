Uma lista duplamente ligada ou lista duplamente encadeada é uma estrutura de dados ligada que consiste de um conjunto de registros sequencialmente ligados 
chamados de nós e é uma extensão da lista simplesmente encadeada. Cada nó contem dois campos, chamados de links ou enlaces, que são referências para o nó anterior 
e para o nó posterior na sequência de nós. Os links anteriores e posteriores dos nós inicial e final, respectivamente, apontam para algum tipo de terminador, 
tipicamente para nulo, para facilitar o percorrimento da lista. Nesta lista o usuário estará cadastrando jogos em uma lista duplamente encadeada. 
Abaixo estão exemplificadas as funções.

Botões:
Inserir - É pedido para entrar com as informações do jogo;
Remover - É pedido para entrar com o nome do jogo a ser removido;
Crescente - Apresenta os elementos da lista em ordem crescente;
Derescente - Apresenta os elementos da lista em ordem decrescente;
Buscar - É pedido para entrar com o nome do jogo a ser buscado;
Limpar tela -Todos os elementos serão removidos;
Retornar - Volta para a tela inicial de seleção de estrutura de dados.
Menu - Arquivo:
Abrir - É pedido para entrar com um arquivo válido para criar a lista;
Salvar - Salva em um arquivo a lista que estiver nessa execução.
Menu - Operações:
É possível realizar as operações igual as presentes nos botões.
Canvas:
Botão esquerdo do mouse: Inserir um aluno ;
Botão direito do mouse: Retirar um aluno.
